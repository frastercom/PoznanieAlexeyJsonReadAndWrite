Создать программу которая считает xml файл
В xml хранятся данные по дням недели, наименование авто и расход топлива за день
1) необходимо вычеслить средний расход за день относительно недели и вывести пользователю
2) необходимо вывести дни недели когда расход превышел среднее значение
3) необходимо вывести дни недели когда расход был ниже среднего

Описание работы с xml https://metanit.com/sharp/tutorial/16.2.php
Как устроены xml https://habr.com/ru/post/524288/

пример xml файла
<?xml version="1.0" encoding="utf-8" ?>
<days>
  <day name="Понедельник">
    <car>Lada</car>
    <liter>21</liter>
  </day>
  <day name="Вторник">
    <car>Lada</car>
    <liter>25</liter>
  </day>
  <day name="Среда">
    <car>Lada</car>
    <liter>19</liter>
  </day>
  <day name="Четверг">
    <car>Lada</car>
    <liter>18</liter>
  </day>
  <day name="Пятница">
    <car>Lada</car>
    <liter>26</liter>
  </day>
</days>

пример вывода:
Средний расход в день: 21,8
Дни с превышение среднего расхода: вторник, пятница
Дни с расходм ниже среднего: понедельник, среда, четверг
